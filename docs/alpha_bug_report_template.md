# Alpha 缺陷记录模板

用途：

- 给 Alpha 人测和复测统一记录格式
- 让后续修复和回归不再依赖聊天上下文回想“当时到底坏在哪”

---

## 单条缺陷模板

```markdown
## [严重度] 用例 ID - 一句话问题标题

- 版本号：
- 账号角色：
- 用例 ID：
- 案例名 / 案例 ID：
- 测试步骤：
- 预期结果：
- 实际结果：
- 截图 / 录屏：
- 备注：
```

---

## 严重度说明

- `P0`：阻断闭环或数据错账
- `P1`：数据不同步或强迷惑
- `P2`：可继续但很别扭
- `P3`：纯视觉细节

---

## 单轮测试结论模板

```markdown
## Round X 测试结论

- 测试版本：
- 测试日期：
- 测试者：
- 覆盖用例：
- 新增 P0：
- 新增 P1：
- 新增 P2 / P3：
- 是否允许进入下一轮：
- 备注：
```
